Experiências profissionais

Como Analista de Dados na área de recuperação de crédito e bacharel em Direito, meu tema central é resolver problemas de negócios. Isso me permite abordar os problemas de negócios não apenas de uma perspectiva técnica, mas também legal. Tenho a responsabilidade de analisar dados complexos para auxiliar na tomada de decisões estratégicas. Uma parte significativa do meu trabalho envolve a criação de dashboards informativos e interativos usando Power BI. Além disso, automatizo processos sempre que possível para aumentar a eficiência e reduzir o tempo gasto em tarefas repetitivas. Isso é feito principalmente através da programação em Python no Visual Code e Jupyter Notebook.

Projetos em Análise de Dados

Painel de Desempenho Empresarial com Power BI

Neste projeto, os conceitos de análise de dados, visualização de dados e design foram usados para criar dashboards no Power BI, visando acompanhar a performance de uma empresa e dos funcionários do ramo financeiro. O resultado final do projeto foi um conjunto de dashboards publicados na web pelo Power BI Service, com atualizações automáticas, e disponibilizados através de um link web. Os dashboards podem ser acessados por qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Power BI Desktop
  • Power BI Web
  • DAX
  • Power Query
  • M
  • DBeaver
  • PostgreSQL
  • Python

Verificador de Integridade e Segurança de Links com Python

Neste projeto, os conceitos de programação em Python e análise de dados foram usados para desenvolver um aplicativo que verifica a integridade e segurança dos links da internet. O resultado final do projeto foi um aplicativo.

As ferramentas utilizadas foram:

  • Python
  • Anaconda
  • Jupyter Notebook
  • Git
  • Visual Code

Desenvolvimento de um Painel Gerencial para Negócio com o Stremlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India. O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Python
  • Anaconda
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • Github
